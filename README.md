# FAPF-TD3 HMUN Network


**Dataset for "Fairness-Aware, Adaptive, and Federated Model-Free Scheduling for Hydrogen-Integrated Multi-Area Urban Power and Traffic Networks"**


__Overview__

This repository contains the datasets used in the research paper titled "Fairness-Aware, Adaptive, and Federated Model-Free Scheduling for Hydrogen-Integrated Multi-Area Urban Power and Traffic Networks" These datasets are crucial for understanding and replicating the test systems applied within the study.
